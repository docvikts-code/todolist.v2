# TaskFlow PWA — Інструкція розгортання

## 📁 Структура файлів
```
taskflow-pwa/
├── index.html       ← головна сторінка
├── manifest.json    ← PWA маніфест
├── sw.js            ← Service Worker (фонові нагадування)
└── icons/           ← іконки для всіх платформ
    ├── icon-72.png
    ├── icon-96.png
    ├── icon-128.png
    ├── icon-144.png
    ├── icon-152.png
    ├── icon-192.png
    ├── icon-384.png
    └── icon-512.png
```

---

## 🚀 Варіант 1 — GitHub Pages (безкоштовно, рекомендовано)

1. Зареєструйся на [github.com](https://github.com)
2. Створи новий репозиторій (наприклад `taskflow`)
3. Завантаж **всі файли** (включно з папкою `icons/`)
4. Перейди в `Settings → Pages → Source: main branch / root`
5. Через 1-2 хв твій додаток буде доступний за адресою:
   `https://твій-username.github.io/taskflow/`
6. Відкрий цю адресу на телефоні → з'явиться кнопка **"Встановити"**

---

## 🚀 Варіант 2 — Netlify (ще простіше)

1. Зайди на [netlify.com](https://netlify.com) → "Add new site"
2. Просто **перетягни папку** `taskflow-pwa` у вікно браузера
3. Netlify автоматично розгорне сайт і дасть HTTPS-посилання
4. Відкрий посилання на телефоні → встанови як додаток

---

## 📱 Як встановити на телефон

### Android (Chrome):
- Відкрий сайт → з'явиться банер "Встановити"
- АБО: меню ⋮ → "Додати на головний екран"

### iPhone (Safari):
- Відкрий сайт у Safari
- Натисни кнопку "Поділитися" (квадрат зі стрілкою)
- Вибери "На початковий екран"

### Windows/Mac (Chrome/Edge):
- Відкрий сайт → у адресному рядку з'явиться іконка встановлення ⊕
- Натисни → "Встановити"

## 🔔 Як працюють нагадування

| Коли | Нагадування |
|------|-------------|
| За 24 години | 📅 Нагадування за добу |
| За 1 годину  | ⏰ Нагадування за годину |
| За 5 хвилин  | 🚨 Термінове нагадування |

**Важливо:** Для роботи нагадувань потрібен HTTPS (GitHub Pages / Netlify надають автоматично).

- **При відкритому додатку**: нагадування спрацьовують через браузерні таймери
- **У фоновому режимі**: Service Worker перевіряє задачі через Periodic Background Sync (Chrome на Android)
- **На iPhone**: сповіщення у фоні потребують iOS 16.4+ і встановленого PWA

---

## ⚙️ Локальне тестування

Для тестування потрібен локальний HTTPS-сервер:

```bash
# Встанови Node.js, потім:
npx serve .

# АБО через Python:
python3 -m http.server 8080
# Потім відкрий http://localhost:8080
```

> ⚠️ Service Worker і сповіщення **не працюють** при відкритті `index.html` файлом напряму (через `file://`). Потрібен сервер.
